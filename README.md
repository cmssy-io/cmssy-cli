# @cmssy/cli

The `cmssy` CLI wires a [Next.js](https://nextjs.org) (App Router) app to a headless
[cmssy](https://www.cmssy.com) workspace - and helps you grow it with blocks.

```bash
npx @cmssy/cli init
```

## Commands

| Command                  | What it does                                          |
| ------------------------ | ----------------------------------------------------- |
| `cmssy init`             | Scaffold a new app or wire cmssy into an existing one |
| `cmssy link`             | Connect an initialized project to a workspace         |
| `cmssy add block <name>` | Scaffold a new block and register it                  |
| `cmssy doctor`           | Diagnose a project's cmssy setup                      |
